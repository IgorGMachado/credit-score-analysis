# Credit Score Analysis

Exploratory Data Analysis project developed using Python and Jupyter Notebook.

## Project Objective
Analyze a customer dataset to understand the factors that influence credit score.

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Scikit-learn
- Imbalanced-learn (SMOTE)

## Project Steps
1. Data cleaning
2. Handling missing values
3. Categorical encoding
4. Correlation analysis
5. Train/Test split
6. Data balancing using SMOTE

## Notebook
The full analysis can be found in the notebook:

`credit_score_analysis.ipynb`
